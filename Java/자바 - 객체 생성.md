# 자바 - 객체 생성

> 자바 객체에 대해 공부하고 정리한 내용입니다.



## 객체 생성과 사용방법

+ ~~~Java
  File file = new File("C:\\Workspace\\JavaPrj\\data.txt");
  //file은 이름, 참조, 레퍼런스 라고 불림
  ~~~



## 스트림 객체 생성방법

+ ~~~Java
  File file = new File("C:\\Workspace\\JavaPrj\\data.txt");
  
  FileOutputStream fout = new FileOutputStream(file);
  fout.write(30);
  fout.flush(); //자바 플랫폼(버퍼)에서 디스크로
  fout.close();
  ~~~
