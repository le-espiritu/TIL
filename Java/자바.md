## JDK

+ #### JDK란?

  + Java Development Kit의 약자를 일컬음.
  + JDK(자바 소프트웨어 개발 키트)는 자바 애플리케이션을 구축하기 위한 플랫폼 구성요소이다.
  + 자바 가상머신(JVM)과 컴파일러, 디버거, 자바 애플릿 및 응용프로그램 개발을 위한 도구들이 포함된다.
